# **App Name**: Blueprint to Automated Gains (BAG)

## Core Features:

- Login-Gated Access: Secure access to the platform using custom tokens (__initial_auth_token).
- Course Navigation: Intuitive navigation through courses, modules, and topics.
- Real-Time Progress Tracking: Visual indicators and real-time updates for tracking course progress in Firestore.
- Admin Panel (User Management): Admin interface for user management: view, search, filter, modify roles, and deactivate users.
- Graphical Course Editor (Placeholder): Placeholder for a graphical course editor: create, edit, reorder (drag & drop), delete courses/modules/topics.
- Content Preview: Allows admins to preview topic content as regular users would see it, pulling data directly from Firestore.
- LLM Content Display: Topic content powered by LLM with reasoning tool available for administrators to include or exclude parts of it in the course generation.

## Style Guidelines:

- Primary color: #826FFF (purple) to represent the core brand identity of Blueprint to Automated Gains. Use consistently across buttons, progress bars, headers, and other UI accents.
- Background color: #E5E1FF, a desaturated tint of the primary color.
- Accent color: #6347FF, analogous to the primary but with higher saturation and a slightly shifted hue.
- Body and headline font: 'Inter' sans-serif font for a minimalist, modern, high-clarity feel.
- Simple, clean icons in #826FFF (purple) to represent different course modules and topic types. Active/completed states indicated with filled circles and green checkmarks.
- Clean and minimalist layout with clear visual hierarchy. Use whitespace effectively to avoid clutter and maintain focus on key elements.
- Subtle animations for expanding/collapsing modules and updating progress indicators, providing a smooth and engaging user experience.
